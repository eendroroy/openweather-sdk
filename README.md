### Installation

*Maven*

```xml
<dependency>
	<groupId>com.github.eendroroy</groupId>
	<artifactId>openweathermap-sdk</artifactId>
	<version>0.1.1</version>
</dependency>
```

*Gradle*
```groovy
dependencies {
    implementation 'com.github.eendroroy:openweathermap-sdk:0.1.1'
}
```
